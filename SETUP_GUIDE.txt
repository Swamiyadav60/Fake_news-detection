================================================================================
FILE DESCRIPTIONS
================================================================================

utils.py (350 lines)
  - preprocess_text(): Cleans text (lowercase, remove URLs/HTML, stopwords, stemming)
  - validate_input(): Ensures input is valid
  - create_vectorizer(): TF-IDF with 5000 features and bigrams
  - Custom exception classes

data_prep.py (280 lines)
  - SyntheticDataGenerator: Creates realistic fake/real samples
  - 20+ templates for each category
  - 1000 fake + 1000 real samples generated
  - Template-based with word bank randomization

train_model.py (280 lines)
  - ModelTrainer: Orchestrates training process
  - Trains Logistic Regression (max_iter=1000)
  - Trains Passive Aggressive Classifier (max_iter=50)
  - Evaluates both models (Accuracy, Precision, Recall, F1)
  - Saves best model to pickle

db_manager.py (300 lines)
  - DatabaseManager: SQLite operations
  - log_prediction(): Store each prediction
  - get_statistics(): Aggregate statistics
  - get_recent_predictions(): History tracking
  - export_to_json(): Data export capability

app.py (400+ lines)
  - Flask application with 6 API routes
  - POST /api/predict: Main prediction endpoint
  - GET /api/stats: Statistics
  - GET /api/history: Prediction history
  - POST /api/retrain: Manual model retraining
  - Automatic initialization and error handling

templates/index.html (900+ lines)
  - Complete responsive dark-themed UI
  - Embedded CSS (no external stylesheets)
  - Embedded JavaScript (no dependencies)
  - Modern design with animations
  - Works on desktop, tablet, mobile

================================================================================
TROUBLESHOOTING
================================================================================

If you see: "ModuleNotFoundError: No module named 'flask'"
  → Run: pip install -r requirements.txt

If you see: "No module named 'nltk' or NLTK data not found"
  → Run: python -c "import nltk; nltk.download('stopwords'); nltk.download('punkt')"

If port 5000 is in use:
  → Edit app.py line 245: change port=5000 to port=5001

If models training is very slow:
  → Normal: Takes 30-90 seconds on first run
  → Subsequent runs: instant startup

If predictions seem inaccurate:
  → Delete *.pkl files and retrain: python train_model.py
